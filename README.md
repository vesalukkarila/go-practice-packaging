# go-practice-packaging 

Pair programming exercise: This repository is for practicing Go (packaging and basic functionality).

## Usage

### Run locally

```sh
go run main.go vesa
```

### Run tests

```sh
go test ./...
```

### Build and run binary 

1. First build it:
    ```sh
    go build -o main
    ```

2. Run the binary:
    ```sh
    ./main vesa
    ```

    (or leave name blank or a empty string `""` to see an error)
